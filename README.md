### 编译&运行
1. 点击Clone或者下载zip获取代码
2. 使用Visual Studio 2015编译（其实应该没什么所谓，只要支持.net 4.0就好了，之前用vs2013和vs2012都编译过）
3. 选择 `x86平台` ，生成目标程序，运行就好了
5. 勾选自己需要的功能就ok了，如果不管用的话可以尝试再次勾选
